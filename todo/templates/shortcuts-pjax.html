The following are keyboard shortcuts:
  • ? — shows this page
  • h — goes home
  • p — goes to the list of Projects
  • i — goes to the Inbox
  • c — goes to the list of Contexts
  • C — goes to Actions without a Context
  • ! — goes to the command-line interface
  • w — goes to the weekly review
  • r — goes to the weekly review
  • d — goes to Download Your Data
  • T — goes to a text-only view (good for searching)
  • t — goes to a true text-only view of all but deleted items
  • I — goes to a true text-only view of incomplete, even if inactive, items
  • s — goes to the search page
  • / — focuses on the first text box on the page
  • f — follows the link in the flash message (the closeable colored box that pops up at the top of the page right below the navigation bar)

Search the web to learn about your web browser's shortcuts for going back and forward in your browser history. Perhaps command-left-arrow and command-right-arrow (or alt if you have no command key).

Another shortcut for searching is to type in /query into the quick capture box in the top navigation bar. Then click on the link that appears (alternatively, press the TAB key to move keyboard focus away from the quick capture box and then use the "f" keyboard shortcut).

When entering an action, you can type in the name of the context as a shortcut for assigning the context. For example, walk the dog @home.

Similarly, when entering an action, you can prefix it with the name of the project and a colon, for example, miscellaneous: buy soymilk @the store.
