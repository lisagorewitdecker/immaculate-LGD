# Sample pyatdl.proto:ToDoList protobuf

Files in this directory are sample protobufs. Using the generated Javascript /
Dart / Python code like `//pyatdllib/core/pyatdl_pb2.py` and
`//pyatdllib/generated_javascript_protobufs/core/pyatdl_pb.js` you can read
these.

For Python, see `//pyatdllib/ui/immaculater.py` and `(cd ../ && make sh ARGS="--database_filename
../example_serialized_binary_protobufs/pyatdl.ToDoList.sample.with.note.binaryproto")`.
That gives you a CLI where you can say `ls -l -R /` or `dumpprotobuf` or
`astaskpaper`.

These were generated by running the classic django webapp and going to the
'Download Your Data' page and clicking on 'Download Your Data'. A few things
changed later via `make sh` after the proto changed.
